#### Task A: 微观世界的概率波——氢原子电子云模拟 (25分)
**背景知识**：
在量子力学中，氢原子基态（$n=1, l=0$）的径向概率密度函数（即在 $r$ 到 $r+dr$ 壳层内找到电子的概率）为：
$$D(r) = \left( \frac{4r^2}{a^3} \right) \exp\left( -\frac{2r}{a} \right)$$
其中 $a = 5.29 \times 10^{-2} \text{ nm}$ 是玻尔半径。由于指数衰减，我们关注 $r \in [0, 0.25\text{nm}]$ 的范围。

**任务要求**：
1. **径向分布分析**：定义函数 $D(r)$，并找到其在指定区间内的最大值 $D_{\max}$。
2. **舍选抽样实现**：
   * 使用 **NumPy 向量化方法** 生成大量候选点 $(r_1, r_2)$，其中 $r_1 \in [0, 0.25]$, $r_2 \in [0, D_{\max}]$。
   * 筛选出满足 $r_2 < D(r_1)$ 的有效径向距离 $r$。
3. **空间投影与可视化**：
   * 随机产生角度 $\theta \in [0, 2\pi]$。
   * 利用坐标变换 $x = r \cos\theta, y = r \sin\theta$ 将径向分布转化为二维平面分布。
   * 生成至少 10,000 个有效点，绘制散点图，观察“电子云”的密度分布。

**物理思考**：
在你的模拟结果中，电子出现概率最高的半径（即 $D(r)$ 的峰值位置）是多少？它与玻尔模型中的轨道半径有什么联系？

#### Task B: 蒙特卡洛积分的三重境界 (25分)
**背景知识**：
计算圆周率 $\pi$ 是蒙特卡洛的入门课。我们将通过计算定积分 $I = \int_0^1 \frac{4}{1+x^2} dx = \pi$ 来对比三种不同的蒙特卡洛策略。

**任务要求**：
1. **境界一：投点法 (Hit-or-Miss)**：
   在单位正方形 $[0,1] \times [0,4]$ 内随机投点 $(x, y)$，统计落在曲线 $f(x) = \frac{4}{1+x^2}$ 下方的比例。
2. **境界二：平均值法 (Sample Mean Method)**：
   利用积分中值定理的随机版本： $I \approx \frac{1}{N} \sum_{i=1}^N f(x_i)$，其中 $x_i$ 服从 $[0, 1]$ 上的均匀分布。
3. **境界三：辅助函数法 (重要性采样 Importance Sampling)**：
   为了减小方差，我们引入一个“形状相似”的辅助函数 $g(x)$ 作为概率密度进行采样。
   * 选择辅助函数 $g(x) = \frac{4-2x}{3}$（它是 $f(x)$ 在 $[0,1]$ 上的线性近似，且已归一化 $\int_0^1 g(x)dx = 1$）。
   * 使用 **逆采样法** 生成符合 $g(x)$ 分布的随机数 $x'_i$。
   * 计算 $I \approx \frac{1}{N} \sum \frac{f(x'_i)}{g(x'_i)}$。
4. **误差分析**：
   使用相同数量的粒子（如 $N=10^5$），对比三种方法的绝对误差。并绘制“误差随 $N$ 变化”的双对数图，观察哪种方法的系数更小。

**💡 提示**：
* **物理直觉**：投点法最直观但浪费了大量“点”的信息；平均值法利用了每一个点的函数值；辅助函数法则通过“在函数值大的地方多投点”进一步优化了效率。
* **逆采样法推导**：若 $g(x) = \frac{4-2x}{3}$，则累积分布函数 $G(x) = \int_0^x g(t)dt = \frac{4x-x^2}{3}$。令 $u = G(x)$，解出 $x = 2 - \sqrt{4-3u}$。

#### Task C: 醉汉求解拉普拉斯方程 (20分)
**背景知识**：
在第 12 周，我们用有限差分法和解稀疏矩阵求出了二维方槽内的电势分布。现在，我们要用“随机游走”来对特定点进行“降维打击”。
设有一个 $50 \times 50$ 的网格区域。边界条件如下：
* 上边界 (Top)：电势 $U = 100$ V
* 下边界 (Bottom)：电势 $U = 0$ V
* 左边界 (Left)：电势 $U = 0$ V
* 右边界 (Right)：电势 $U = 0$ V

**任务要求**：
1. 编写一个二维随机游走函数。让粒子从坐标 $(x_0, y_0) = (25, 25)$ （即正中心）出发，每次等概率向上下左右四个方向移动一格（ $\Delta x = 1, \Delta y = 1$）。
2. 当粒子碰到任意边界时，停止游走，并记录该边界的电势值。
3. 释放 5,000 个粒子，求它们撞击边界电势的算术平均值。
4. 改变起始点，求坐标 $(25, 40)$（靠近上边界）的电势，看看结果是否符合物理直觉。

**💡 提示**：
* 不需要构建任何矩阵！只需要一个 `while` 循环跟踪粒子的 `(x, y)` 坐标。
* 中心点 $(25, 25)$ 的理论解析解可以通过对称性直接猜出来，你可以用它来校验你的代码是否写对了。

### Lab2_Bonus (30分) - 让时间倒流——蒙特卡洛模拟热传导方程
**背景知识**：
对于一维热传导方程（抛物型 PDE） $\frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2}$，当我们在 FTCS 显式格式中取网格比 $r = 0.5$ 时，公式化简为 $u_i^{n+1} = 0.5 u_{i-1}^n + 0.5 u_{i+1}^n$。这意味着 $t$ 时刻某点的温度，等于粒子在空间上左右随机游走，向过去的时间（ $t=0$）追溯时，落在初始温度分布上的期望。

**任务要求**：
1. 设一维金属棒长度 $L = 1.0$，划分为 100 个网格。
2. **初始条件**：中间一段（ $0.4 \le x \le 0.6$）温度为 $100^\circ\text{C}$，其余部分为 $0^\circ\text{C}$。
3. 我们想知道经过 $T = 500$ 个时间步后，位置 $x = 0.5$（中心点）的温度。
4. **算法实现**：从 $(x=0.5, t=500)$ 出发，让粒子在空间上以 0.5 的概率向左或向右走，同时时间步减 1（时间倒流）。当时间步退回 $t=0$ 时，读取该空间位置的初始温度。
5. 模拟 10,000 个粒子，求平均温度。

**思考**：
相比于第 11-12 周我们用 FTCS 算整个时空网格，这种蒙特卡洛方法有什么独特的优势？
*(提示：如果我们只需要知道某一个特定点、特定时刻的温度，蒙特卡洛方法可以实现“局部求解”，而不需要像差分法那样把整个矩阵和历史时间步全部算一遍，这在极高维度的金融 PDE（如期权定价）中是极其强大的特性。)*
